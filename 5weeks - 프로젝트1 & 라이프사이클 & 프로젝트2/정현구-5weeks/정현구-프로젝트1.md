
| 대상                 | 위치/범위                  | 주요 속성                               | 디자인 효과                                     |
| :------------------- | :------------------------- | :-------------------------------------- | :---------------------------------------------- |
| **`body`**           | 문서 전체                  | `padding: 20px`                         | 화면 가장자리와 콘텐츠 사이에 여유 공간 생성    |
| **`.App`**           | `class="App"` 요소         | `width: 400px`, `margin: 0 auto`        | 전체 컨텐츠를 400px 너비로 고정하고 중앙에 배치 |
| **`.App > section`** | `.App` 바로 아래 `section` | `background`, `border`, `border-radius` | 연한 회색의 둥근 모서리 카드 형태 디자인        |
| **`margin-bottom`**  | 각 `section` 사이          | `10px`                                  | 섹션들 사이의 세로 간격 유지                    |

## 이제 버튼 클릭했을 때 증가시키거나 감소시킬려면?

> 특정 값을 변경시켰을 때, 변경된 값을 컴포넌트가 즉시 화면에 렌더링할려면 값을 `state`로 만들어야함.
>
> > 그래야 state값이 변경됨과 동시에 화면이 리렌더링 되면서 바로바로 반영이 됌.

## state를 어떤 컴포넌트에 만들어야 하는지?

우선 3가지 후보

1. 현재 카운터를 렌더링하는 Viewer 컴포넌트
2. 카운터 값을 실질적으로 변경시키는 Controller 컴포넌트
3. 부모 컴포넌트인 App 컴포넌트

> 정답은 3번  
> 이유는? : Viewer와 Controller 컴포넌트는 서로 부모 자식 관계가 아니기에 어떤 값도 서로 공유하지 못하기 때문에

### 예를 들어 Viewer 컴포넌트에 state를 만들 경우

- Viewer 컴포넌트에 있는 setCount라느 함수를 Controller 컴포넌트에 전달해줘야 되는데 불가능하다.  
  왜냐하면, 컴포넌트 간 데이터를 주고 받을 때 props를 이용하는데 props는 오직 부모 자식관계만 사용 가능하다.

- Controller로 만들 시에도 Count state를 Viewer 컴포넌트에 주는 것도 불가능

![](https://velog.velcdn.com/images/hyunguu/post/25c2b662-554f-4d7d-bfe2-713f19a06ec7/image.png)

- 결국 위와 같은 계층 구조로 부모인 App 컴포넌트에 생성하여 만들어야 함.

## State Lifting (State 끌어 올리기)

### State Lifting이란?

- State Lifting은 여러 자식 컴포넌트가 동일한 데이터에 반응해야 할 때, 그 상태를 공통 부모 컴포넌트로 올리는 과정

### 핵심 코드 분석

```javascript
function App() {
  const [count, setCount] = useState(0); // 1. 공통 상태 생성

  const onClickButton = (value) => {
    setCount(count + value); // 2. 상태를 변경하는 함수 정의
  };

  return (
    <div className="App">
      <section>
        {/* Viewer에게는 데이터만 전달 */}
        <Viewer count={count} />
      </section>
      <section>
        {/* Controller에게는 데이터를 바꿀 수 있는 권한(함수) 전달 */}
        <Controller onClickButton={onClickButton} />
      </section>
    </div>
  );
}
```

```javascript
const Controller = ({ onClickButton }) => {
  return (
    <div>
      {/* 부모가 보내준 onClickButton 함수를 호출하며 인자 전달 */}
      <button onClick={() => onClickButton(1)}>+1</button>
    </div>
  );
```
